# GreenMail Testcontainers for .NET

This package provides a simple way to spin up a disposable [GreenMail](https://greenmail-mail-test.github.io/greenmail/) instance using [Testcontainers for .NET](https://dotnet.testcontainers.org/).  
It is intended for integration testing scenarios where a temporary mail server is required.
